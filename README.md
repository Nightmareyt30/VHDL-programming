# \# Computer Organization and Architecture Lab (VHDL)

# 

# This repository contains VHDL implementations of various digital circuits and computer organization concepts developed as part of the \*\*Computer Organization and Architecture (COA) Laboratory\*\* coursework.

# 

# \## 📚 Overview

# 

# The objective of this repository is to provide VHDL-based designs for fundamental digital components and architectural building blocks commonly studied in COA and Digital Design courses.

# 

# The programs can be simulated and synthesized using standard VHDL tools such as:

# 

# \- Xilinx Vivado

# \- Xilinx ISE

# \- ModelSim

# \- GHDL

# \- Quartus Prime

#

# \## 🚀 Getting Started

# 

# \### Prerequisites

# 

# \- VHDL Simulator/Synthesis Tool

# \- Basic understanding of VHDL and Digital Logic Design

# 

# \### Running a Simulation

# 

# 1\. Open your preferred VHDL tool.

# 2\. Create a new project.

# 3\. Add the required VHDL source files.

# 4\. Add the corresponding testbench.

# 5\. Compile the design.

# 6\. Run the simulation and verify the waveform outputs.
